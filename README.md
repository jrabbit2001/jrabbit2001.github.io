이성록 교수님 퇴임식을 초대합니다.
* 일시 : 2022년 12월 16일 (금) 15시 ~ 18시
* 장소 : 한국복지대학교 

# 초대의 말

학생들의 든든한 아버지가 되어주신 한국복지대학교 이성록 교수님이
정년퇴임을 하시게 되었습니다.
바쁘시더라도 참석해주시어 자리를 빛내주시기 바랍니다.
